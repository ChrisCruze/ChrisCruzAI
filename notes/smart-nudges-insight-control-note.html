Insight & Control

Insight & Control Tower Blueprint

Updated September 15, 2025

Mission

Provide Cuculi teams with a unified visual system to review every smart nudge by recipient, event, and time, enabling rapid QA, confident experimentation, and executive-ready reporting.

Core Views

  • • User Timeline: Chronological feed of sends, suppressions, and overrides.
  • • Event Pulse: Seat inventory, RSVP velocity, and message bursts per table.
  • • Calendar Heatmap: Weekday/hour patterns exposing best send windows.
  • • Channel Health: Delivery, latency, and bounce metrics from Twilio/OneSignal/Mailchimp.

Governance Levers

  • • Real-time alerting for delivery dips or duplicate sends.
  • • Role-based access and audit logs for overrides.
  • • Change history for every YAML trigger edit.
  • • Export APIs that power the Behavioral Evidence Pack.

Dashboard Stack

Data Pipeline

MongoDB change streams feed a warehouse sync every 5 minutes. dbt models power fact tables segmented by user, event, and time grain.

Visualization

Looker governs executive views; Metabase supports ad-hoc queries. Both inherit metric definitions from shared YAML.

Alerting

PagerDuty notifies Ops when send volumes deviate ±30% vs. forecast, while Slack digests flag experiments needing review.

Implementation Checklist

  1. Map every nudge type to ownership teams and define dashboard metric owners.
  2. Create Looker Explores for user, event, and calendar slices with drill-through to raw logs.
  3. Automate QA queries that cross-check Twilio/OneSignal/Mailchimp delivery receipts.
  4. Embed dashboard links directly into the Trigger Matrix for in-context reviews.

Operator Moves

  • • Host twice-weekly “mission control” standups using this dashboard before publishing new experiments.
  • • Share highlights with marketing so copy tweaks mirror high-performing scarcity or reciprocity runs documented in the Evidence Pack.
  • • Align with engineering on schema changes to keep MongoDB trigger metadata rich enough for analytics.
  • • Reference the Four-Pillar Strategy when proposing new views.
